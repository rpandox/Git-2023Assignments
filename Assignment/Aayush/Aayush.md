## <p align='center'>Learned Commands</p>

1.  `git init`

    **Description** :

        It is used to initialize git repository

    **Examples**:

        $ git init

2.  `git add <file_name>`

    **Description**:

        It is used to add files or changes in the working directory to the staged area.

    **Examples**:

        $ git add .gitignore hangman.c

3.  `git commit -m "Commit label`

    **Description**:

        It is used to commit the files or changes in the working directory.
